Non-Destructive Evaluation of High-Voltage Module Using Computed Tomography

A sealed defense module with a recurring failure and no visible cause — diagnosed without opening it.

The challenge

A high-voltage defense module was showing recurring failures with no identifiable external cause. The root cause had to be identified non-destructively, precisely enough to revise the manufacturing process for all subsequent units.

What I did

Led a CT-based failure and root cause analysis using a GE Phoenix v|tome|x scanner. The module was placed on a rotation table and irradiated with a cone-beam X-ray source, producing a magnified 3D reconstruction of the internal geometry. Analyzed the resulting volume data in VGStudio MAX, mapping the three PCBs and transformer inside the sealed housing, identifying the anomaly site, and characterizing the defect geometry. Findings fed directly into a revised manufacturing and inspection protocol.

How performance was evaluated

Root cause confirmed against physical evidence from the CT reconstruction. Process revision implemented on the production line. Reliability improvement tracked across subsequent units. Sample remained intact throughout the analysis, preserving the evidence chain.
